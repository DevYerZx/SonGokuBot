### ⚙️ Instalación en Termux

```bash
cd && termux-setup-storage
apt-get update -y && apt-get upgrade -y
pkg install -y git nodejs ffmpeg imagemagick yarn
git clone https://github.com/DevYerZx/SonGokuBot.git && cd SonGokuBot
yarn install
npm install
npm update
npm start
